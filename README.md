
# 👥 GroupMong – Web Project

### 👨‍💻 Built by SereyodamChek & Team

---

## 📌 About This Project

**GroupMong** is a **responsive web project** built with HTML, CSS, and multimedia assets.

This project was created as a team assignment to demonstrate frontend development skills, creative design, and collaborative workflow using Git & GitHub.

---

## 🚀 Key Features

* 🎨 Clean and modern web design
* 🖼️ Image gallery & visual content integration
* 🎬 Embedded video support for rich media experience
* 📱 Responsive layout optimized for multiple screen sizes
* 🗂️ Organized folder structure for easy maintenance
* 🌐 Pure static site — fast loading & no backend required

---

## 🛠️ Tech Stack

* 🌐 **HTML5** – Semantic page structure
* 🎨 **CSS3** – Styling, layouts, and responsive design
* 🖼️ **Media Assets** – Optimized images & video content
* 📁 **File Organization** – Modular structure for scalability

---

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/SereyodamChek/groupmong.git
cd groupmong
```

### 2. Open in browser

Simply open `test1.html` in your preferred web browser:

```bash
# Option 1: Double-click test1.html in your file explorer
# Option 2: Use a local server for best results

# With Python
python -m http.server 8000

# With Node.js
npx serve .
```

Then visit: [http://localhost:8000](http://localhost:8000)
